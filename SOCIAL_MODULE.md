# Социальный модуль (Мини-блог)

## 📋 Обзор

Социальный модуль позволяет пользователям публиковать заметки о финансовой грамотности, делиться опытом и общаться в комментариях.

## 🎯 Возможности

### Для пользователей

- **Публикация постов** с заголовком, содержимым и изображением
- **Выбор видимости**: публичный (видно всем) или приватный (видно только автору)
- **Лайки** - возможность отметить понравившиеся посты
- **Комментарии** - обсуждение постов с другими пользователями
- **Лента постов** с фильтрацией (все, публичные, мои)
- **Редактирование** своих постов и комментариев
- **Аватар профиля** с возможностью загрузки изображения
- **Описание профиля** (bio) - краткая информация о себе

### Технические особенности

- **Медиа-файлы**: загрузка аватаров и изображений постов
- **Пагинация** постов на бэкенде
- **Real-time обновления** лайков и комментариев
- **Адаптивный дизайн** для мобильных устройств

## 📡 API Endpoints

### Посты

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/blog/posts/` | Список постов |
| POST | `/api/v1/blog/posts/` | Создать пост |
| GET | `/api/v1/blog/posts/{id}/` | Получить пост |
| PUT | `/api/v1/blog/posts/{id}/` | Обновить пост |
| DELETE | `/api/v1/blog/posts/{id}/` | Удалить пост |
| POST | `/api/v1/blog/posts/{id}/like/` | Лайкнуть пост |
| POST | `/api/v1/blog/posts/{id}/unlike/` | Убрать лайк |
| GET | `/api/v1/blog/posts/{id}/comments/` | Комментарии к посту |
| POST | `/api/v1/blog/posts/{id}/add_comment/` | Добавить комментарий |

### Профиль

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/auth/profile/` | Получить профиль |
| PUT | `/api/v1/auth/profile/` | Обновить профиль (avatar, bio, currency) |

## 📸 Использование

### Создание поста

1. Перейдите на страницу **Блог**
2. Нажмите кнопку **"Создать пост"**
3. Заполните заголовок и содержимое
4. Прикрепите изображение (опционально)
5. Выберите видимость (публичный/приватный)
6. Нажмите **"Опубликовать"**

### Редактирование профиля

1. Перейдите в **Профиль**
2. Нажмите на иконку камеры на аватаре для загрузки изображения
3. Заполните поле **"О себе"**
4. Нажмите **"Сохранить"**

## 🔐 Безопасность

- Только авторизованные пользователи могут создавать посты
- Пользователи могут редактировать/удалять только свои посты
- Приватные посты видны только автору
- Публичные посты видны всем пользователям

## 📊 Модели данных

### Post (Пост)
- `author` - автор поста (ForeignKey на User)
- `title` - заголовок (200 символов)
- `content` - содержимое (текст)
- `visibility` - видимость (public/private)
- `image` - изображение (опционально)
- `likes` - количество лайков
- `created_at`, `updated_at` - даты

### Comment (Комментарий)
- `post` - пост (ForeignKey на Post)
- `author` - автор (ForeignKey на User)
- `content` - текст комментария
- `created_at`, `updated_at` - даты

### Profile (Профиль пользователя)
- `user` - пользователь (OneToOne на User)
- `avatar` - изображение профиля
- `bio` - описание (500 символов)
- `currency` - предпочитаемая валюта

## 🎨 Компоненты фронтенда

### PostCard
Основной компонент для отображения поста:
- Информация об авторе с аватаром
- Заголовок и содержимое
- Изображение (если есть)
- Кнопки лайка и комментариев
- Список комментариев с возможностью добавления

### PostModal
Модальное окно для создания/редактирования:
- Форма с заголовком и содержимым
- Загрузка изображения с предпросмотром
- Выбор видимости (публичный/приватный)
- Валидация полей
