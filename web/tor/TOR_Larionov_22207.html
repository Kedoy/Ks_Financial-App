Техническое задание проекта "Финансовый помощник"

I. Общие сведения о проекте

Название проекта: Финансовый помощник
Цель проекта: Создание веб-сайта для учета личных финансов с элементами аналитики и геймификации.

II. Страничное содержание сайта

0. Основные элементы для всех страниц сайта

Шапка сайта - "логотип"; кнопки гиперссылки на вход/регистрацию
Левая секция - навигация между главной страницей, игрой, тестом и разделом "О проекте"
Футер - контактная информация, лицензия

1. Главная страница сайта

Центр - таблица для данных о расходах со столбцами: дата, название, категории, сумма.
Кнопки управления таблицей: управление категориями (по нажатию откроется окно с выбором добавить/удалить категорию),
управление расходами (окно с добавить/удалить запись о расходе), кликабельная сортировка по столбцам (дате/сумме/категории).
Правая секция - сводка - аналитика и статистика (диаграммы, рекомендации).

Для неавторизованных пользователей таблица и сводка недоступны, на них кнопка "Войти, чтобы продолжить".

2. Страница мини-игры "Кешбекни!"

В стиле волка с яйцами: ловить падающие по желобам чеки.
Используя canvas и javascript создайте анимированную мини-игру.
Система отслеживания лучших результатов.

Для авторизованных пользователей планируется сохранение результатов на сервер. Неавторизованные пользователи могут играть в гостевом режиме.

3. Страница теста

Если данные о расходах отсутствуют - предложение заполнить таблицу расходов.
Необходимо использовать различные варианты задания ответов:
выбор одного варианты, множественный выбор, выпадающий список, ввод числа, ввод текста, вопросы на соответствие.
(реализация с динамической генераций теста по списку вопросов/ответов/правильных ответов/варианта вывода из какого либо источника(массив,объект,json и т.д.))

Для неавторизованных пользователей: предложение авторизации для доступа к тесту.

4. Страница с информацией о проекте

Центр - наполнение текстом, картинками и подписями к ним.
Правая секция - форма обратной связи через электронную почту.

5. Страница с аналитикой и статистикой

Центр - график расходов по дням, диаграммы с расходами по категориям, максимальный/минимальный расход за день.
Настройка отображаемых данных: какие именно категории будут показаны на диаграмме?

Для неавторизованных пользователей предложение авторизации для просмотра данных.

6. Страница с формой регистрации/входа в аккаунт

Оформленное окно ввода логина+пароля/окно регистрации.
